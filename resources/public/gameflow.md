# Spyfall Game Flow

## Overview
Spyfall is a social deduction game where players work together to identify a hidden spy among them. The spy must deduce the secret location without being caught.

## Game Setup
1. **Player Count**: 3-8 players recommended
2. **Location Selection**: Randomly select a location card from the deck
3. **Role Assignment**:
   - All players except one receive the same location card
   - One player receives a "Spy" card and does not know the location
4. **Timer**: Set a 10-minute timer for the discussion phase

## Discussion Phase (10 minutes)
1. Players take turns asking questions about the location
2. Questions should be open-ended and location-specific
3. The spy must blend in by giving plausible answers without revealing they don't know the location
4. Players can ask follow-up questions to probe suspicious answers

## Voting Phase
1. When time runs out or players are ready, discussion ends
2. Each player votes for who they think is the spy
3. Players cannot vote for themselves
4. The player with the most votes is accused

## Resolution
1. **If the accused is the spy**:
   - The spy can attempt to guess the location
   - If correct, spy wins
   - If incorrect, non-spies win
2. **If the accused is not the spy**:
   - The spy wins
   - The wrongly accused player is eliminated (optional variant)

## Winning Conditions
- **Non-spies win** if they correctly identify and accuse the spy
- **Spy wins** if they avoid detection or correctly guess the location when accused

## Tips for Play
- Ask specific questions about the location
- Pay attention to inconsistencies in answers
- The spy should ask questions that reveal information without giving away their ignorance
